## 2. Опис бібліотек

### 2.1 Pandas

**Призначення:** Маніпуляція та аналіз даних

Pandas — найпопулярніша бібліотека Python для роботи зі структурованими (табличними) даними. Вона надає структури даних DataFrame та Series, які дозволяють ефективно обробляти великі набори даних.

**Основні можливості:**
- Читання/запис даних з CSV, JSON, Excel, SQL та інших форматів
- Фільтрація, сортування та групування даних
- Обробка пропущених значень
- Статистичний аналіз та агрегація
- Об'єднання та злиття наборів даних

**Приклад з проекту:**

```python
import pandas as pd

# Читання CSV файлу
df = pd.read_csv("comptagevelo2010.csv")

# Перетворення дати та витяг місяця
df["Date"] = pd.to_datetime(df["Date"], format="%d/%m/%Y")
df["Month"] = df["Date"].dt.month

# Групування за місяцем та обчислення суми
monthly = df.groupby("Month")["Berri1"].sum()

# Фільтрація даних
high_scholarship = df[df['scholarship'] > 2000]

# Описова статистика
print(df.describe())
```

### 2.2 Matplotlib

**Призначення:** Візуалізація даних

Matplotlib — базова бібліотека для побудови графіків у Python. Вона надає MATLAB-подібний інтерфейс для створення статичних, анімованих та інтерактивних візуалізацій.

**Основні можливості:**
- Лінійні графіки, стовпчасті діаграми, кругові діаграми, гістограми
- Точкові діаграми та діаграми розмаху
- Налаштування осей, легенд, заголовків та підписів
- Підтримка підграфіків для кількох графіків
- Експорт у різні формати (PNG, PDF, SVG)

**Приклад з проекту:**

```python
import matplotlib.pyplot as plt

# Стовпчаста діаграма
plt.figure(figsize=(10, 6))
plt.bar(monthly.index, monthly.values, color="steelblue")
plt.xlabel("Місяць")
plt.ylabel("Велосипедисти")
plt.title("Щомісячний велосипедний трафік (2010)")
plt.xticks(range(1, 13))
plt.show()

# Лінійний графік з кількома серіями
plt.plot(years, ukraine_data, marker="o", label="Україна")
plt.plot(years, usa_data, marker="s", label="США")
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

# Кругова діаграма
plt.pie(populations, labels=years, autopct="%1.1f%%")

plt.show()
```

### 2.3 NumPy

**Призначення:** Чисельні обчислення

NumPy — фундаментальна бібліотека для наукових обчислень у Python. Вона забезпечує підтримку багатовимірних масивів та матриць, а також широкий набір математичних функцій.

**Основні можливості:**
- N-вимірні масиви (ndarray)
- Транслювання для поелементних операцій
- Операції лінійної алгебри
- Генерація випадкових чисел
- Перетворення Фур'є
- Інтеграція з іншими бібліотеками (Pandas, Matplotlib)

**Приклад з проекту:**

```python
import numpy as np

# Створення рівномірно розподілених значень
x = np.linspace(0.001, 5, 1000)

# Математичні операції над масивами
y = 5 * np.cos(10 * x) * np.sin(3 * x) / np.sqrt(x)

# Використання з Matplotlib
plt.plot(x, y, linewidth=2.5)
plt.title("Y(x) = 5*cos(10x)*sin(3x)/√x")
plt.show()
```

### 2.4 NLTK (Natural Language Toolkit)

**Призначення:** Обробка природної мови

NLTK — комплексна платформа для створення Python-програм, що працюють з даними природної мови. Вона надає інструменти для токенізації, стемінгу, тегування, парсингу тощо.

**Основні можливості:**
- Токенізація тексту (розбиття на слова/речення)
- Видалення стоп-слів
- Стемінг та лематизація
- Тегування частин мови
- Доступ до текстових корпусів (Gutenberg, Brown тощо)
- Аналіз частотності

**Приклад з проекту:**

```python
import nltk
from nltk.corpus import gutenberg, stopwords
from nltk.tokenize import word_tokenize
from collections import Counter

# Завантаження необхідних ресурсів
nltk.download("gutenberg", quiet=True)
nltk.download("punkt", quiet=True)
nltk.download("stopwords", quiet=True)

# Завантаження та токенізація тексту
text = gutenberg.raw("chesterton-ball.txt")
words = word_tokenize(text.lower())

# Підрахунок частоти слів
word_counts = Counter(words)
top_10 = word_counts.most_common(10)

# Видалення стоп-слів
stop_words = set(stopwords.words("english"))
clean_words = [w for w in words if w not in stop_words and w.isalpha()]
```

## 5. Висновок

Кожна бібліотека має своє призначення в екосистемі обробки даних Python:

- **Pandas** найкраще підходить для маніпуляції табличними даними і часто є першим вибором для задач аналізу даних
- **Matplotlib** надає комплексні можливості візуалізації та добре інтегрується з іншими бібліотеками
- **NumPy** є основою для чисельних обчислень, пропонуючи високопродуктивні операції над масивами
- **NLTK** пропонує повний набір інструментів для задач обробки природної мови

Комбінація цих бібліотек дозволяє вирішувати широкий спектр задач обробки даних — від простих файлових операцій до складного статистичного аналізу та візуалізації.
